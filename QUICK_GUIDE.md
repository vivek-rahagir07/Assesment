# TalentCalibrate — Quick Guide

Welcome to TalentCalibrate. This guide gives new users a fast walkthrough of key screens and workflows. The app also includes an interactive Quick Guide that highlights UI elements and explains their purpose.

## Main concepts

- Workspace: A container for templates, candidates, and analytics for a team or hiring cycle.
- Template (Rubric): A structured evaluation form with weighted criteria and checklist questions.
- Panel Mode: A session where multiple evaluators grade the same candidate and scores are averaged.
- Candidate Scorecards: Individual evaluations that result in an overall calibrated percentage.

## Quick tour (what you'll see)

1. Create a Workspace
   - Click "Create New Workspace" on the landing screen to set up a dedicated space for your team.

2. Evaluation Rubrics (Dashboard)
   - The "Evaluation Rubrics" tab lists templates. Click any template to start a live interview or view existing scorecards.

3. Customize Rubrics
   - Use the "Customize Rubrics" tab to edit or create templates. Adjust criteria names, weights, and default questions.

4. Calibration & Analytics
   - The "Calibration & Analytics" tab shows aggregate statistics, performance buckets (Top/Mid/Low), and export options (CSV/Excel/PDF).

5. Panel Mode
   - Create a panel session from the dashboard. Share the session code with your team so everyone can join and evaluate the same candidate.

6. Chat Assistant
   - Open the floating assistant (bottom-right) for quick help, suggested questions, and workspace insights.

7. Exports and Reports
   - Use export buttons in the analytics view to download candidates and scorecards for further analysis.

## Best practices

- Keep rubrics concise (6–8 criteria) and set weights so the totals add up to 100%.
- Use Panel Mode for calibration meetings — compare individual evaluator ratings and align on standards.
- Save drafts frequently; the app supports offline storage and automatic syncing when back online.

## Troubleshooting

- If your changes don't appear, try refreshing. The app uses IndexedDB with multi-tab persistence; if sync issues persist, check network status.
- For export failures, ensure the browser allows file downloads and popups.
